# 图标说明

此目录需要放置 tabBar 图标文件，共需要 6 个图标：

## 图标列表

| 文件名 | 尺寸 | 说明 |
|--------|------|------|
| home.png | 81x81px | 首页未选中 |
| home-active.png | 81x81px | 首页选中 |
| job.png | 81x81px | 职位未选中 |
| job-active.png | 81x81px | 职位选中 |
| my.png | 81x81px | 我的未选中 |
| my-active.png | 81x81px | 我的选中 |

## 图标要求

- 尺寸：81 x 81 像素（实际显示会被压缩为 48px）
- 格式：PNG
- 颜色：
  - 未选中状态：#999999 (灰色)
  - 选中状态：#1890ff (蓝色，与导航栏颜色一致)
- 背景：透明

## 快速获取图标

你可以从以下资源获取免费图标：
1. [阿里巴巴图标库](https://www.iconfont.cn/) - 搜索 "首页"、"职位"、"我的"
2. [IconPark](https://iconpark.oceanengine.com/) - 字节跳动开源图标库
3. 使用微信小程序官方图标库

## 临时方案

在开发阶段，你可以使用在线图标链接（在 app.json 中修改路径），但上线前必须替换为本地图片。
